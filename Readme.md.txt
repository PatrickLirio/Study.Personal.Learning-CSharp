Namespace - is used to logically organize classes, structs, interfaces, enums, and delegates.
Namespaces in C# can be nested, meaning one namespace can contain other namespaces.

using - is a keyword that allows you to use the types (classes, structs, etc.) defined inside a namespace without writing the full namespace name.

System - is a built-in .NET Core namespace that contains fundamental classes and types such as Console, String, and Math.

Comment - is an annotation in a source code with the intention to give a programmer a readable explanation of the code.It is ignored by compilers when compiling your code. ex. //, /**/.
